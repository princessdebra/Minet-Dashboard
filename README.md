# Minet-Dashboard
## Overview
A comprehensive analytics dashboard for insurance brokers, providing insights into won/lost accounts, revenue trends, market share, and key performance metrics. Built with React, Recharts, and Tailwind CSS.

## Features
### Dashboard Landing
KPI Cards: 8 key metrics with animated values (won/lost accounts, revenue, win rate, etc.)

Interactive Charts:

Won vs. Lost Accounts (Composed Bar Chart)

Revenue Trend (Line Chart)

Top Product Lines (Vertical Bar Chart)

Sales Pipeline Table: Opportunity tracking

Quick Navigation: Links to detailed pages

### Won Accounts
Monthly acquisition trends

Premium band distribution

Underwriter performance

Client segment analysis

### Lost Accounts
Churn reason breakdown

Competitor analysis

Projected vs. actual losses

Win-back campaign tracking

### Market Share
Product line distribution

Client segment penetration

Geographical performance

Industry vertical analysis

### Revenue Walk
Monthly/quarterly trends

Money in/out breakdown

Key revenue drivers

Growth opportunities

## Technical Stack
Frontend: React 18, Vite

Charts: Recharts

Styling: Tailwind CSS

Icons: Lucide React

State Management: React Hooks

API: REST endpoints (local development)
